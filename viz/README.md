# T-PA-G 三层目标—公理—要求体系 · 可视化

本目录提供《三层目标—公理—要求体系 v2》的**五种可交互前端可视化**，便于从不同视角查看 T→PA→G 的追溯关系，并点击节点查看文字说明。

**开源与在线发布**：规则、要求与实现方式（GitHub Pages 配置、URL 约定等）见 **[开源与在线发布说明.md](./开源与在线发布说明.md)**。

## 如何打开

- **在线访问（推荐）**：若仓库已配置 GitHub Pages（Source 为分支 `main` 的 `/docs`），在浏览器打开：  
  `https://<owner>.github.io/<repo>/rule/viz/`  
  无需本地部署。详见 [开源与在线发布说明.md](./开源与在线发布说明.md)。
- **本地预览**：在项目根目录执行 `npx serve docs/rule/viz` 或进入本目录执行 `python -m http.server 8080`，浏览器访问对应地址后打开 `index.html`。
- **本地文件**：直接双击 `index.html` 用浏览器打开（部分浏览器对 `file://` 加载同目录 `data.js` 可能有限制，若图表不显示可改用上述本地服务）。

## 五种方案

| 方案 | 说明 |
|------|------|
| **方案一：三列流式图** | T \| PA \| G 三列，节点为矩形，贝塞尔曲线连线，类似 Sankey 的层级流。 |
| **方案二：神经网络式** | 三层水平排列（上 T、中 PA、下 G），节点为圆，直线连接，类似 NN 传播图。 |
| **方案三：径向同心圆** | T 内圈、PA 中圈、G 外圈，极坐标排布，连线经中心弯曲。 |
| **方案四：可折叠树** | 按 T→PA→G 的树形列表，点击 ▶/▼ 展开/收起子节点。 |
| **方案五：力导向图** | 节点可拖拽，T/PA/G 分色，边拉近、节点互斥的简单力导向布局。 |

所有方案均支持：**点击节点 → 弹出该节点的定义/表述与说明**（与文档对应）。

## 文件说明

- `index.html`：入口页，顶部 Tab 切换五种方案。
- `data.js`：图数据（T/PA/G 节点与 T→PA、PA→G 边），供各方案共用。
- `viz1-flow.js` ~ `viz5-force.js`：五种方案的渲染与交互逻辑。

数据来源：`docs/rule/三层目标—公理—要求体系_v2.md` 中的 T→PA、PA→G 映射与各节点摘要。
